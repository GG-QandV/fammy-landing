Код (вставить в export metadata)
Найти в файле существующий export const metadata = {...} или создать, если его нет (вариант 2):

tsx
import type { Metadata } from 'next';

export const metadata: Metadata = {
  title: 'Fammy.pet - Pet Nutrition Validator',
  description: 'Validate homemade pet recipes instantly. Check ingredients for toxicity, balance nutrients, and keep your dog or cat healthy.',
  openGraph: {
    title: 'Fammy.pet - Pet Nutrition Validator',
    description: 'Validate homemade pet recipes instantly. Check ingredients for toxicity, balance nutrients, and keep your dog or cat healthy.',
    url: 'https://fammy.pet',
    siteName: 'Fammy.pet',
    images: [
      {
        url: 'https://fammy.pet/og-image.jpg',
        width: 1200,
        height: 630,
        alt: 'Fammy.pet - Pet Nutrition Validator',
      }
    ],
    locale: 'en_US',
    type: 'website',
  },
  twitter: {
    card: 'summary_large_image',
    title: 'Fammy.pet - Pet Nutrition Validator',
    description: 'Validate homemade pet recipes instantly. Check ingredients for toxicity, balance nutrients, and keep your dog or cat healthy.',
    images: ['https://fammy.pet/og-image.jpg'],
  },
};

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  );
}

Расположение в файле
text
1. Импорты (import)
2. export const metadata = {...}  ← ВСТАВИТЬ ЗДЕСЬ
3. export default function RootLayout(...)

Если metadata уже существует
Заменить существующий блок metadata полностью или добавить только секцию openGraph внутрь существующего объекта.
